# 🚀 VARAOS – AI Device Control Landing Page

A futuristic, high-performance landing page built with **React 19 + Vite 8 + Tailwind CSS v4**, inspired by next-gen AI systems and privacy-first architecture.

---

## ✨ Features

* 🌌 Dark futuristic UI (neon + glassmorphism)
* 🔐 Privacy-first design aesthetics
* ⚡ Fully responsive layout
* 🧠 AI-inspired visual storytelling
* 🔄 Orbital animation system (Hero section)
* 🎯 Modular component architecture
* 🚀 Lightning-fast performance (Vite)

---

## 🛠️ Tech Stack

* **React 19**
* **Vite 8**
* **Tailwind CSS v4 (CSS-first config)**
* **React Router v7**
* **React Icons v5**
* **JavaScript (ES6+)**

## 🎨 Tailwind CSS v4 Setup

This project uses **Tailwind v4 CSS-first configuration**:

* Theme variables defined using `@theme`
* Custom utilities via `@layer utilities`
* No `tailwind.config.js`
